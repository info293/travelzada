## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - React components for different sections
- `public/` - Static assets

## Sections

1. **Header** - Navigation and sign up button
2. **Hero** - Main banner with CTA
3. **Explore by Category** - Travel category cards
4. **How It Works** - 3-step process explanation
5. **Explore by Activity** - Activity-based exploration
6. **Trending This Month** - Popular destinations
7. **Summer Deals** - Special offers banner
8. **Explore India by State** - State-wise destinations
9. **Testimonials** - Customer reviews
10. **Why Travelzada** - Key features
11. **Trending Destinations** - International destinations
12. **Blog Section** - Travel guides and articles
13. **Footer** - Links and social media

## Build

To create a production build:

```bash
npm run build
```
